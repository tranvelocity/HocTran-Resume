How to work with sample applications

Put the sample application into the server

  • Please put whole of ShotviewSample folder and below to the PHP server.

Configure the sample applications

  • Please edit ShotviewSample/phpcommon/define.php, line 10, put your 'SDK auth. password' at SDK_AUTH_PASSWORD.

Check the sample

  • Please access to ShotviewSample/index.html using the smartphone.

Development Environment

  • We have checked with PHP7.
  • You need curl on PHP.

Client Program Operating Environment

  • iOS:
    • iPhone, iPad with iOS 10 or above. iPod Touch is out of scope.
    • Only Safari is eligible for operation guarantee.
  • Android:
    • 5.0 or above
    • Only newest chrome is eligible for operation guarantee.
